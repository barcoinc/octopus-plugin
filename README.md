# OCTOPUS

セミナー台本 Deep Analysis Engine — ターゲットの信念を見抜き、刺さる台本を作る

## 概要

OCTOPUS はテンプレートを埋めて台本を量産するツールではありません。
ターゲットが**何を信じていて、何に矛盾を抱えていて、何が刺さるか**を見抜き、
その分析の上にセミナー台本を作るAIパートナーです。

## セットアップ

1. プラグインをインストール
2. 「連携」をクリック
3. APIキーを入力してログイン
4. チャットで「始めたい」と話しかける

## 機能

- **情報入力** — 会話であなたの商品・ターゲット情報を収集
- **ターゲット分析** — 信念・矛盾を深掘りして仮説を生成
- **セミナー台本作成** — 素材選定→アウトライン→本文執筆
- **フィードバック** — セミナーの反応を記録、次回の精度向上
